Q1: Why might we consider using Git?

Answer: We might considering using Git when working on projects where 
multiple people need to contribute. It allows them to work on individual 
parts without breaking anything that someone else needs to work.

Q3: What is the difference between Merging and Rebasing?

Answer: Merging combines two branches keeping the commit structure. 
Rebasing is different because it rewrites commit history by applying 
commits in order to create a cleaner commit history.
